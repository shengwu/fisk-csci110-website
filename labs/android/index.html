Lab 10: Android

Prequisites

  • Android Studio downloaded

Assignment

Please complete the Building Your First App tutorial.

Then, repeat the "Building a Simple User Interface" and "Starting Another Activity" steps. Create five buttons that send the user to five different activities, each with a bio of one of your favorite athletes, celebrities, or musicians.

Each button should take the user to an activity just for that person's bio. Do the same thing that the tutorial did when displaying text in an activity. Except this time, the text is a pre-set string.

When you're done, put your entire project in a zip folder and upload to Canvas.
