Lab 7: CodeFights

CodeFights is one of my favorite websites to practice programming. You can do daily challenges, compete in tournaments against other players, challenge individuals to 1v1 matches, do time trials against engineers at specific companies, and more.

Here's an example of a problem that involves bit shifting. The problem is described on the left. Write your answer in the code pane in the upper right, then click Run Tests to test your code. In this screenshot, I've put in a wrong answer so the tests failed. You get a chance to look at each failing test in the bottom right:

After writing a correct answer and verifying it with the visible test cases, you can click Submit to test it on the complete test case set. With any luck, your code will pass all of the tests:

Your assignment

Work with a partner to solve all eight challenges in the Code Arcade's Intro Gates. Each person should create his or her own CodeFights account, solve all eight problems, and submit his or her own code to Canvas by copying and pasting it into the text box in Canvas. Work only with your partner.
