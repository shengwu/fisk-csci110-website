Part 3: faces

Change Face.java to draw three separate faces across the screen. First, write a function that lets you draw a face in a specific location. Then add code to your main method to call the function and draw the face in different locations.

You must use a function in this part. No credit will be given for programs where the face-drawing code was simply copy and pasted three times.

After you create a function, your program should look something like this. If you added a line of code to the main function "drawFace(g);" so that you had two in a row, it would draw two faces on top of each other. Are there parameters you can add to draw the face in different places?

import java.awt.*;

public class Face {
    static void drawFace(Graphics g) {
        g.setColor(Color.BLACK);
        g.drawOval(10, 30, 100, 100);
        g.setColor(Color.BLUE);
        g.fillOval(30, 60, 20, 20);
        g.fillOval(70, 60, 20, 20);
        g.setColor(Color.RED);
        g.drawLine(40, 100, 80, 100);
    }

    public static void main(String[] args) {
        DrawingPanel panel = new DrawingPanel(320, 180);
        Graphics g = panel.getGraphics();

        drawFace(g);
    }
}
