Running JUnit in the lab

To use JUnit in the lab, we have to specify the classpath manually since we can't change environment variables on the lab computers. First download jUnit and download hamcrest, Then run the following command to compile your files and tests (replacing "YOUR_USERNAME" with your fisk username and My.java and MyTest.java with the names of your files):

javac -classpath .;C:\Users\YOUR_USERNAME\Downloads\hamcrest-core-1.3.jar;C:\Users\YOUR_USERNAME\Downloads\junit-4.12.jar My.java MyTest.java

To run a test file called MyTest, run this command (replacing "YOUR_USERNAME" with your fisk username and MyTest with the class name of your test):

java -classpath .;C:\Users\YOUR_USERNAME\Downloads\hamcrest-core-1.3.jar;C:\Users\YOUR_USERNAME\Downloads\junit-4.12.jar org.junit.runner.JUnitCore MyTest
